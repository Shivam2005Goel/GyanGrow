## ğŸš€ Getting Started Locally

### Prerequisites

- Node.js (v18+)
* Active Gemini API Key
* Uploadcare Public Key

### Installation

1. **Clone & Install**

   ```bash
   git clone https://github.com/vatsalgupta2025-creator/VITGROWW.git
   cd VITGROWW
   npm install
   ```

2. **Environment Configuration**
   Create a `.env.local` globally setting up your AI and File keys:

   ```env
   NEXT_PUBLIC_GEMINI_API_KEY=your_key_here
   NEXT_PUBLIC_UPLOADCARE_PUBLIC_KEY=your_key_here
   ```

3. **Deploy Development Server**

   ```bash
   npm run dev
   ```

   *GyanGrow will compile and launch on `http://localhost:3000`.*
